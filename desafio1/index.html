Estrutura de um arquivo .html

Um arquivo html é estruturado através de tags, existem 2 tags básicas que toda página html tem:

  • <head></head> que contém informações funcionais sobre a página, como a tag <title></title> e tags <meta>. Ela não é visível quando a página é carregada.
  • <body></body> que contém todo o conteúdo visível da página.

Além dessas duas tags, existem tagas que ajudam a organizar o código de maneira limpa além de ajudar na acessibilidade:

  • <header></header> ela é usada para delimitar a parte de cima do código, onde fica a barra de navegação.
  • <nav></nav> ela é usada para delimitar a barra de navegação.
  • <main></main> ela é usada para delimitar o conteúdo principal da página.
  • <section></section> ela é usada para delimitar uma seção específica dentro da página.
  • <aside></aside> ela é usada para colocar um conteúdo do lado de outro.
  • <footer></footer> ela é usada para colocar os elementos de rodapé da página.

Tags de formulário <form></form>

É uma tag utilizada para abrir uma seção de formulário. Dentro dessa tag podemos ter as tags <input></input> para inserir dados e a tag <label></label> para dar nome aos inputs.










Além do input, podemos utilizar a tag <select></select> em conjunto com a tag <option></option> para ter outra forma de receber os dados de um usuário.

E no final de tudo, podemos ter um botão para enviar essas informações do formulário.
